Windows

1. 安装Qt5

  1. 点击进入Qt官网下载Qt5,这里我下载的是最新版的Qt5.11,windows平台.

tips:由于文件比较大,最好右键->复制链接地址->打开迅雷,用迅雷下载

  1. 下载完成后,双击安装

  2. 有三个地方要注意

    • 让你注册登录的时候,可以Skip跳过
    • 自行修改安装路径最好是英文的
    • 选择安装组件的时候,要Select All
    • 然后就是一路Next,最后Install了.

      tips:博主在安装过程中,会弹出一个Installer Error的警告,本人是直接点Ignore的,忽略此警告

  3. 最后,点击Finish完成

  4. 自此,Qt5的安装完毕

2. 配置Qt5的环境

2.1 安装windbg

默认情况下是没有调试器的，必须手动下载

下载调试器windbg,在安装过程中,只勾选Debugging Tools for Windows,其他不选

2.2 验证

  1. 打开 Qt Creator，工具->选项->构建和运行->编译器,进入编译器部分，可以看到 Qt 已经自动检测出来了，不需要手动配置

  1. 而构建套件(Kit)中,已经自动检测出来调试器了

3. VS2017配置

3.1 安装插件Qt VIsual Studio Tools

  1. 打开VS2017->工具->扩展和更新

  1. 联机->搜索框输入Qt->Qt Visual Studio Tools->下载,然后就会自动安装了,安装完成会提示重启VS2017

  1. 重启vs2017后,就会看到Qt VS Tools菜单项了

3.2 插件配置

Qt VS Tools->Qt Options->Add->
